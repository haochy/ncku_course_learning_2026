# 🚀 Cyber-Scroller | 橫向捲軸遊戲

這是一款基於 HTML5 Canvas 和 JavaScript 開發的經典「超級馬力歐」風格橫向捲軸遊戲。

![Game Preview](file:///home/hao/.gemini/antigravity/brain/4f64aaf4-6dfa-4179-b6bb-c49ad9d5f65e/game_start_screen.png)

## 🌟 遊戲特色

- **自定義像素資產**：使用 AI 生成的獨特賽博龐克風格角色與場景。
- **流暢的動態捲軸**：攝影機跟隨主角移動，呈現廣闊的關卡體驗。
- **經典遊戲機制**：
  - **跳躍與重力**：具有物理感的操控體驗。
  - **敵人對抗**：可以踩踏敵對無人機來擊敗它們。
  - **數據節點蒐集**：蒐集發光的金幣來獲取積分。
- **完整的遊戲流程**：包含開始畫面、遊戲中 HUD、失敗重開以及勝利關卡。

## 🎮 控制說明

- **方向鍵 左/右** (Left/Right)：移動角色。
- **方向鍵 上** (Up)：跳躍。
- **空白鍵** (Space)：重開（在結束畫面時）。

## 🛠️ 技術細節

- **Core**: HTML5, Vanilla JavaScript, CSS3
- **Rendering**: Canvas API
- **Design**: Cyberpunk Neon Aesthetics

## 🚀 如何執行

1. 將此儲存庫複製或下載到本地。
2. 直接點擊 `index.html` 即可在任何現代瀏覽器中開始遊戲！
